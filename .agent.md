# 项目概览：auto-video

## 项目定位

**海外热点股市自动化工作流** — 每日自动抓取海外股市新闻，生成中文文案，并推送到多个平台。

## 文件结构

```
auto-video/
├── scripts/
│   └── foreign_market_digest.py   # 核心脚本（~26KB）
├── finance_daily/
│   ├── notifiers/                  # 通知模块（目录为空）
│   ├── templates/                  # 模板目录（目录为空）
│   └── __pycache__/               # Python 缓存（config, cli, fetchers）
├── .github/workflows/
│   └── foreign-market-content.yml # GitHub Actions 工作流
└── readme.md
```

## 核心功能（`foreign_market_digest.py`）

| 功能 | 说明 |
|------|------|
| 数据抓取 | Google News RSS，覆盖美股、欧股、亚太 |
| 主题分类 | 8大主题（利率、科技、财报、通胀、能源等） |
| 评分排序 | 按地区权重、时效性、来源权威性打分 |
| 推送渠道 | 飞书 Webhook、微信 PushPlus、微信公众号草稿 |
| 无第三方依赖 | 纯标准库实现 |

## GitHub Actions

- **定时触发**：每天 UTC 07:30（北京时间 15:30）
- **产物上传**：`output/` 目录作为 Artifact

## 注意点

- `finance_daily/` 目录有 Python 缓存（`config`, `cli`, `fetchers`），但对应源文件不存在 — 可能是模块化重构的遗留痕迹
- `notifiers/` 和 `templates/` 目录为空，说明重构尚未完成
